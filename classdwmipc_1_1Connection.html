Detailed Description

The DWM IPC connection class used to initiate a connection with DWM's IPC socket and send/receive messages.

void dwmipc::Connection::subscribe ( const Event  ev)

Subscribe to the specified DWM event. After subscribing to an event, DWM will send dwmipc::MESSAGE_TYPE_EVENT messages when the specified event is raised. Subscribing to an already subscribed event will not throw an error.

Parameters

Exceptions

◆ unsubscribe()

void dwmipc::Connection::unsubscribe ( const Event  ev)

Unsubscribe to the specified DWM event. After unsubscribing to an event, DWM will no longer send dwmipc::MESSAGE_TYPE_EVENT messages for the specified event. Unsubscribing to an already unsubscribed event will not throw an error.
