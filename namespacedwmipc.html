◆ recv_message()

std::shared_ptr< Packet > dwmipc::recv_message ( int  sockfd,
bool  wait 
)

Receive any incoming messages from the specified socket. This is the main helper function for attempting to read a message from DWM and validate the structure of the message. It absorbs EINTR, EAGAIN, and EWOULDBLOCK errors.

Parameters
sockfdThe file descriptor of the socket to receive the message from
waitShould it wait for a message to be received if a message is not already received? If this is false, the file descriptor should have the O_NONBLOCK flag set.
Returns
A received packet from DWM
Exceptions
NoMsgErrorif no messages were received
HeaderErrorif packet with invalid header received
EOFErrorif unexpected EOF while reading message
